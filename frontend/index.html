데이터 소스: 해양환경공단(KOEM) 측정망 + 기상청(KMA) ASOS + 국립수산과학원(NIFS) 적조속보. 모델 출력은 1/3/7일 후 적조 발생 확률(0~100%).

지표 설명

수온 (SST, 해수면 수온, °C)
바다 표층의 물 온도. SST는 Sea Surface Temperature의 약자로 일반적으로 말하는 "수온"과 같은 값이다. 코클로디니움·카레니아 같은 적조 원인 와편모조류는 20~28℃ 범위에서 급증하므로 여름철 수온 상승은 적조의 1차 신호다.
염분 (Salinity, psu)
해수 1kg당 녹은 염류의 양(g). 정상 해수는 약 32~34psu이며, 장마·태풍 직후 담수 유입으로 떨어진다. 저염분(28~31psu)은 영양염 유입과 함께 적조 종 증식을 촉진하는 조건이 된다.
클로로필-a (Chl-a, µg/L)
식물플랑크톤이 가진 광합성 색소. 즉 해수의 식물플랑크톤 총량을 대리하는 지표다. 평상시 1~3 µg/L에서 적조 발생기에는 10~50 µg/L까지 치솟으므로 적조 임박을 가장 직접적으로 보여준다.
기타 입력 (DIN/DIP/DO/pH + 기상)
질소·인 영양염(DIN/DIP)은 플랑크톤 먹이, 용존산소(DO)·pH는 수질 상태, 기상청 ASOS의 기온·풍속·일사·강수는 성층화·혼합과 광합성에 영향을 준다.

AI 모델

HistGradientBoostingClassifier × 3 horizons
scikit-learn의 히스토그램 기반 그래디언트 부스팅 분류기. +1일·+3일·+7일 적조 발생 여부를 각각 예측하는 3개 모델을 따로 학습한다. 결측치를 그대로 학습에 사용할 수 있고 수만 건 표본에서도 빠르게 수렴해 본 데이터셋(KOEM 분기측정·NIFS 라벨)에 적합하다.
학습 데이터·기간
KOEM 7종(수온·염분·pH·DO·DIN·DIP·chl-a) + KMA ASOS 4종(일사·풍속·기온·강수) + 7/30일 이동평균 + 계절 변수. NIFS 적조속보를 (region, date) 라벨로 사용. 2023-01-01 이전을 학습, 이후를 검증으로 분리(chronological holdout)하고 walk-forward CV 5폴드로 미래 누설을 차단했다.
검증 성능 (AUROC)
+1일 0.85 / +3일 0.85 / +7일 0.87 (holdout). walk-forward CV 평균 0.74~0.78. 어종 출현 확률은 별도 휴리스틱(SST·염분·계절)으로 산출하며 향후 KOSIS 어획 통계 기반 ML로 교체 예정이다.
